# Sistema de Cálculo de IMC em Java

Este projeto consiste em um sistema simples desenvolvido em Java com o objetivo de aplicar conceitos fundamentais de lógica de programação, controle de fluxo e interação com o usuário via terminal.

## Funcionalidades

- Menu interativo no terminal
- Cálculo automático do IMC (Índice de Massa Corporal)
- Classificação do IMC conforme padrões de saúde
- Repetição de operações sem reiniciar o programa
- Validação de entradas do usuário

## Conceitos aplicados

- Estruturas de decisão (`if`, `else if`, `switch`)
- Estruturas de repetição (`while`, `do-while`)
- Entrada de dados com `Scanner`
- Manipulação de dados numéricos
- Organização lógica de fluxo
- Validação de dados

## Fórmula do IMC

IMC = peso / (altura * altura)

- Ter o Java instalado (JDK 8 ou superior)
- Variáveis de ambiente configuradas (`java` e `javac`)
